# Utility Bill Automation System

A Python desktop application that automates utility bill data extraction from PDF invoices and updates Excel workbook templates while preserving formulas and reducing manual entry errors.

## Features

- Extracts meter-level data from PDF bills
- Parses readings, usage, multipliers, and charges
- Cleans and validates extracted data
- Updates Excel templates using openpyxl
- Preserves workbook formulas and summary logic
- Desktop GUI using Tkinter
- Validation and mismatch detection

## Tools & Technologies

- Python
- Tkinter
- pdfplumber
- openpyxl
- Regex
- PyInstaller

## Workflow

PDF Bill → Text Extraction → Parsing → Validation → Excel Update

## Disclaimer

This repository contains sanitized/sample logic only. No real billing data or internal documents are included.
